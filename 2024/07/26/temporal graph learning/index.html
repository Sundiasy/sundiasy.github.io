基础时序图神经网络调研

​ 引用自Medium的文章Temporal Graph Learning in 2023 ，时态图网络（TGN）将消息传递神经网络（MPNN）推广到时态图中。 它们通过引入节点MEM来实现这一目的，节点MEM代表节点在给定时间内的状态，是节点过去交互行为的压缩表示。 每当两个节点发生交互时，它们就会互相发送信息，然后用来更新它们的MEM。 在计算节点的嵌入时，会对节点的时间邻居进行额外的图聚合，同时使用原始节点特征和该时间点的记忆。 下面是 TGN 的计算示意图。

​ 这个部分在之前学习TGN的报告中有叙述，最基础的TGN训练过程主要过程是进行以下循环：MEM更新(RNN等)->图聚合(注意力等)->当前时间loss计算并与之前的total loss累加后更新权重->更新MEM…

最新进展调研

​ 此部分大部分参考来源于Medium文章Temporal Graph Learning in 2024 ，同时包括之前的学习与调查。主要包括数据集Benchmark的发布、链路预测的进展、时序知识图谱、对抗攻击等。这里只挑选关于舆情早期预测相关的方向汇报。

时序图Benchmark

​ 我认为舆情早期预测最好的方式是自建数据集，以解决实际问题。在最新发展的一项综述文章里，我找到了文章中关于当前最好的几种测试时序图神经网络能力的benchmark（Temporal Graph Benchmark, TGB）。TGB 提供了一个 pypi 软件包，用于自动下载和处理来自五个不同领域的九个数据集，这些数据集具有多达 7200 万条边和 3000 万个时间戳。

​ 在Benckmark的dynamic node property prediction（节点属性预测）任务上，网上作者发现目前的TGN方法没有启发式算法的表现优异，对此在节点预测或可解释任务上，可以从此角度出发解释与创新。（*Here, we use the Normalized Discounted Cumulative Gain of the top 10 items (NDCG@10) to compare the relative order of the predicted items to that of the ground truth. Interesting, we found that single heuristics outperform existing TG models and this highlights the need for more models focusing on node level tasks in the future.*）

时空图学习Spatiotemporal Graphs

​ 时空图被用来表示具有固定拓扑和随时间变化的节点特征的图，通常以与定期采样的观测相对应的离散时间步长变化。时态图认为将动态节点特征视为时间序列，将边视为观测序列之间的函数依赖关系。这不同于舆情早期预测的要求，舆情早期预测主要是变化的拓扑结构以及固定的节点特征。

​ 时空图可用于交通预测，如固定的拓扑结构（道路之间的联系，不会随时间变化），具体的工作在Graph Deep Learning for Time Series Forecasting 论文中有详细介绍。

因果感知时态图学习Causality-Aware Temporal Graph Learning

​ 因果感知时态图是在无向图中考虑图结构的时间拓扑。一个简单的例子，两个无向边*(a,b)和(b,c)分别出现在时间t₁和t2处，(a,b)和(b,c)分别使用无向图的边连接，三个节点间的关系是无向的。如果(a,b)发生在(b,c)之前，则a可以通过经过b的时间相关路径对c*产生因果影响，即建立了专门的时间拓扑图描述因果感知关系，如下图所示，因果关系不具有橙色线条代表的时间拓扑：

​ 具体表示来说，图中的每一条边都是长度为1时间相关路径，构成了时间图因果拓扑的一阶模型。一节模型忽略了边的时间顺序，因此在一阶模型中只计算了每个边出现的频率。二阶模型描述了因果特性，简单地将一阶图中的边替换为二阶图中的节点，即将边（a，b）和（b，c）分别变成节点“a→b”和“b →c” ，用长度为2的时间相关路径表示，即边*(a→b, b→c)表示a通过b因果影响c* 。

​ 更为一般的，可以将因果时间图推向更高阶k，从而产生用于时间图因果拓扑的 k阶De Bruijn图模型。静态路径的中心性（例如接近度或介数）可能会给出关于时态图中节点的真正重要性的误导性结果。因果序列时序图可用于解决基于时序的中心性预测问题（论文Using Causality-Aware Graph Neural Networks to Predict Temporal Centralities in Dynamic Graphs ）。

可解释的时序图神经网络方法

​ 有学者探寻可解释性通过找到影响结果的节点集。论文Explaining Temporal Graph Models through an Explorer-Navigator Framework 使用explorer-navigator架构，使用搜索树的方式，从给定大小的时间图子集中查找影响节点集合。

Deep Graph Library库

​ Deep Graph Library (dgl.ai) 是一个用于处理图神经网络（GNN）的 Python 库，包括图构建操作、图神经网络模型构建操作、高效计算操作等各种对于图数据结构的高效操作。在我的省级大创RGCN模型的构建中，我使用了DGL库，深深地感受到了其使用的方便性。

​ 在DGL库中可以找到目前经典的GNNs模型如GCN，GAT，RGCN，TGN等复现。使用DGL库可以结合Pytorch、Tensorflow等，结合张量复现经典的图神经网络模型，提高运算效率。
