DYREP: LEARNING REPRESENTATIONS OVER DYNAMIC GRAPHS

Introduction

回答两个问题

  • What can serve as an elegant model for dynamic processes over graphs?
  • How can one leverage such a model to learn dynamic node representations that are effectively able to capture evolving graph information over time

回答第一个问题，需要构建

  1. 拓扑演化：其中节点和边的数量预计会随着时间的推移而增长（或缩小），从而导致图中的结构变化;
  2. 以及节点交互：其涉及可能在结构上连接或可能不连接的节点之间的活动。对这些非线性演化的动态过程之间的交错依赖关系进行建模是推进动态图的正式模型的关键下一步。

回答第二个问题，目前现存的时序图方法包含两类

  • 一类是D-TDGs，分立时间提取快照，这些方法倾向于保留（编码）非常有限的结构信息并且在非常粗略的级别捕获时间信息，这导致快照之间的信息丢失并且缺乏捕获细粒度时间动态的能力。
  • 连续时间方法，在更细的时间粒度上对演化进行建模

我们提出，任何动态图必须最低限度地表示为两个基本过程在不同的时间尺度上演变的结果

  • Association Process：改变了图的结构，并导致节点之间的长期信息交换
  • Communication process：与节点之间（不一定连接）的活动相关，导致节点之间的临时信息流

在文章中作者做了事情：我们提出了一种新的动态图表示学习框架DyRep，通过上面表达的潜在中介过程来模拟两个观察过程的交错进化，并随着时间的推移有效地学习更丰富的节点表示。我们的框架摄取动态图形信息的形式的关联和通信事件随着时间的推移，并更新节点表示当这些事件发生时。我们建立一个两个时间尺度的深度Point process捕获两个观测过程的连续时间细粒度时间动态。我们进一步用一个深度归纳表示网络来参数化时间点过程的条件强度函数，该网络学习函数来计算节点表示。最后，我们通过设计一种新的时间注意机制来耦合我们框架的结构和时间组件，该机制使用学习的强度函数来诱导邻域节点的时间注意。这允许捕获随着时间的推移控制节点表示的高度交织和非线性动态。我们设计了一个高效的无监督训练过程，用于我们框架的端到端训练。我们展示了一致的和显着的改进，在两个现实世界的动态图的动态链接预测和时间预测的任务的最先进的代表性基线。我们进一步提出了一个广泛的定性分析，通过嵌入可视化和消融研究，以辨别我们的框架的有效性。

背景

Point Process

随机点过程是一种随机过程，其实现由时间上的离散事件t1、t2、….时间点过程就是这样一个随机过程，它可以等价地表示为计数过程N（t），它包含了直到时间t的事件的数目。描述时间点过程的一般方法是通过条件强度函数λ（t），即给定先前事件的事件发生率的随机模型。形式上，λ（t）dt是在小窗口[t，t + dt）中观察到事件的条件概率，$\lambda(t) \mathrm{d} t:=\mathbb{P}[ $ event in $ [t, t+\mathrm{d} t) \mid \mathcal{T}(t)]=\mathbb{E}[\mathrm{d} N(t) \mid \mathcal{T}(t)]$其中$$ T ( t ) = t _ { k } | t _ { k } < t$$是t之前的历史。类似地，对于t > tn且给定历史T = t1，…，tn，我们将在[tn，t]期间无事件发生的条件概率刻画为$$ S(t \mid \mathcal{T})=\exp \left(-\int_{t_{n}}^{t} \lambda(\tau) \mathrm{d} \tau\right) $$，这被称为过程的生存函数的报告。此外，事件在时间t发生的条件密度被定义为$$ f ( t ) = \lambda ( t ) S ( t ) .$$。强度λ（t）通常用于捕获感兴趣的现象-常见形式包括泊松过程、霍克斯过程、自我纠正过程。

NOTATIONS AND DYNAMIC GRAPH SETTING

包括两种

Event Observation：事件Communication和Association Process都是以时间窗口[t0，T]上图G上的节点之间观察到的二元事件dyadic events的形式实现的，并按时间排序。对于时间t的任何类型的事件，我们使用以下形式的规范元组表示：$ e = ( u , v , t , k )$其中u，v是涉及到事件的两个节点。t表示事件的时间。k ∈ {0，1}，我们使用k = 0表示来自拓扑Association Process（关联）的事件，使用k = 1表示来自节点Communication Process（通信）的事件。图中的边仅通过拓扑事件（Association process）出现，而交互事件（Communication process）不贡献它们。因此，k表示与分别生成拓扑（网络的动态）和交互事件（网络上的动态）的过程相关联的尺度（演化速率）的抽象。然后，我们将窗口[0，T]中按时间排序的P个观察事件的全集表示为$$ O = \left{ ( u , v , t , k ) _ { p } \right} _ { p = 1 } ^ { p }$$。$$ t _ { p } \in R ^ { + } , 0 \leq t _ { p } \leq T .$$

Node Representation：令$$ z ^ { v } \in R ^ { d }$$表示节点v的d维表示。随着表示随时间的推移而演变，我们将它们限定为时间的函数：$$ z ^ { v } ( t )$$节点v在时间t涉及v的事件之后更新的表示。我们使用$z ^ { v } ( t )$来表示节点v在t之前的最近更新嵌入。

Dynamic Graph Setting

我们的框架观察图的演化作为一个事件流O，因此任何新的节点将始终被观察到作为这样的事件的一部分。这将导致从数据中获得的节点的自然排序。由于我们的方法是归纳的，我们从不学习特定于节点的表示，而是学习函数来计算节点表示。由于我们的方法是归纳的，我们从不学习特定于节点的表示，而是学习函数来计算节点表示。在这项工作中，我们只支持网络的增长，即我们只对节点和结构边的添加进行建模，并将删除保留为未来的工作。此外，对于模型的一般描述，我们将假设图中的边没有类型，节点没有属性，但我们在附录B中讨论了如何使用我们的模型来适应这些特征的细节。

PROPOSED METHOD: DYREP

定义local、global effects。当在两个节点之间观察到事件时，信息从一个节点的邻域流向另一个节点，并相应地影响节点的表示。Communication event（交互）只在两个节点之间传播本地信息，但Association event会改变拓扑结构，从而具有更大的全局影响。

MODELING TWO-TIME SCALE OBSERVED GRAPH DYNAMICS

使用时间点过程表示communication and association events。定义节点u和节点v在时间t时的条件密度函数为
$$
\lambda _ { k } ^ { u , v } ( t ) = f _ { k } ( g _ { k } ^ { u , v } ( t ) )
$$
$$ \overline { t }$$代表仅仅早于当前事件发生的时间，附加定义如下：
$$
g _ { k } ^ { u , v } ( t ) = \omega _ { k } ^ { T } \cdot \left[ z ^ { u } ( \overline { t } ) ; z ^ { v } ( \overline { t } ) \right]
$$
可训练参数和concat。

定义fk需要考虑因素1. 强度必须是正的。2. communication and association process被包含在不同的时间尺度上，于是使用改进的Softplus定义：
$$
f _ { k } ( x ) = \psi _ { k } \log ( 1 + e x p ( x / \psi _ { k } ) )
$$
$ \psi _ { k }$是可训练参数，对应于由相应过程产生的事件的速率。

LEARNING LATENT MEDIATION PROCESS VIA TEMPORALLY ATTENTIVE REPRESENTATION NETWORK

节点更新需要基于以下原则，

  • Self-Propagation：自传播可以被认为是一个最小的组成部分的动态管理一个单独的节点的演变。节点在嵌入空间中表示相对于其先前位置（例如，特征集）而不是以随机方式演进。

  • Exogenous Drive：一些外生力可以在时间间隔期间（例如，在涉及该节点的两个全局事件之间）平滑地更新节点的当前特征。

  • Localized Embedding Propagation：事件中涉及的两个节点形成了一个临时（通信）或永久（关联）路径，用于将信息从一个节点的邻域传播到另一个节点。这对应于通过参与事件的另一节点的二阶邻近度节点的影响（参见附录A的图形描述）。

定义为：
$$
\mathbf{z}^{v}(t_{p})=\sigma(\underbrace{\mathbf{W}^{struct}\mathbf{h}{struct}^{u}(\bar{t{p}})}{\text{Localized Embedding Propagation}}+\underbrace{\mathbf{W}^{rec}\mathbf{z}^{v}(\bar{t{p}^{v}})}{\text{Self-Propagation}}+\underbrace{\mathbf{W}^{t}(t{p}-\bar{t_{p}^{v}})}_{\text{Exogenous Drive}},
$$

1)节点u充当将信息从$${\mathcal{N}}_{u}$$传递到节点v的桥梁，因此v通过u以聚合形式接收信息。2)当u的每个邻居将其信息传递给v时，节点u中继的信息由聚合函数管理，该聚合函数由u与其邻居的通信和关联历史参数化。

TEMPORALLY ATTENTIVE AGGREGATION

对于一个给定的节点，并不是所有的邻居都是同样重要的，因此以某种加权的方式从每个邻居捕获信息变得非常重要。我们提出了一种新的基于时间点处理的注意机制，该机制使用时间信息来计算节点之间的结构边的注意系数。然后，这些系数用于计算嵌入传播所需的聚合量（$\mathbf{h}_{struct}$）。

为了正式捕获不同邻居的影响差异，我们提出了一种新的基于条件强度的注意层，该注意层使用矩阵S来诱导共享注意机制来计算邻居的注意系数。
$$
q_{ui}(t)=\frac{\exp(\mathcal{S}{u i}(\bar{t}))}{\sum{i^{\prime}\in\mathcal{N}{u}(t)}\exp(\mathcal{S}{u i^{\prime}}(\bar{t}))}
$$
$$ q_{ui} $$表示在时间t的邻居i的注意权重，因此它是时间上演变的量。然后，通过采用如下的跨邻居的关注聚合机制，使用这些关注系数来计算节点u的聚合信息$$\mathbf{h}{struct}^u(\hat{\bar{t}})$$：
$$
\mathbf{h}{struct}^{u}(\bar{t}):=:\max\left(\left{\sigma\left(q_{ui}(t)\cdot\mathbf{h}^{i}(\bar{t})\right),\forall i\in\mathcal{N}_{u}(\bar{t})\right}\right)
$$

其中，$$\mathbf{h}^{i}(\bar{t}):=:\mathbf{W}^{h}\mathbf{z}^{i}(\bar{t})+\mathbf{b}^{h}:\mathrm{and}:\mathbf{W}^{h}:\in:\mathbb{R}^{d\times d}:\mathrm{and}:\mathbf{b}^{h}:\in:\mathbb{R}^{d}$$是控制u的每个邻居传播的信息的参数$\mathbf{z}^{i}(\bar{t})\in\mathbb{R}^{d}$是node i最近的embedding。

Construction and Update of S

我们构造一个随机矩阵S（在前面的部分中用于参数化注意力）来捕获复杂的时间信息。在初始时间点t = t0，我们直接从A（t0）构造S（t0）。构建$$\mathcal{S}{v}(t{0}):\mathrm{as:}\mathcal{S}{vu}(t{0}):=:0:\mathrm{if}:(v:=:u:\mathrm{or}\mathbf{A}{vu}(t{0}):=:0)$$且满足条件：$$\mathcal{S}{vu}(t{0}):=:\frac{1}{|\mathcal{N}{v}(t{0})|}:\mathrm{if}:\mathcal{N}{v}(t{0}):=:{u:::\mathbf{A}{uv}(t{0}):=:1}.$$

再接受到事件$$o=(u,v,t,k)$$后，我们根据k的观测值对A和S进行更新。A仅针对关联事件被更新（k=0，结构改变）。因此，S的值仅在两种情况下更新/激活：

  1. 当前事件是已经具有结构边的节点之间的相互作用$$(\mathbf{A}_{uv}(t)=1\mathrm{and}k=1)$$
  2. 当前事件是Association event（k = 0）。给定节点u的邻域，b表示每个边缘的背景（基本）注意力，其是基于邻域大小的均匀注意力。

更新注意力在以下操作中：对于情况(1)，使用事件的强度来更新对应的S条目的注意力。对于情况（2），重复与（1）相同的操作，但也随着邻域大小的增加，调整具有其他邻域的边缘的背景注意力（通过$$b-b^{\prime}$$，$$b$$和$$b^{\prime}$$分别是新的和旧的背景注意力）。从数学观点来看，这种更新类似于标准的时间点过程公式，其中来自b的项用作背景注意力，而λ可以被视为基于内源强度的注意力。
