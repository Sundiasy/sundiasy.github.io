Deep Learning for Dynamic Graphs: Models and Benchmarks

本文记录了典型的图神经网络baseline和benchmarks，第二节简要介绍了静态图的表示学习，提供了用于定义动态图问题的一般定义和方法。第三节正式介绍了动态图的表示学习，而第四节和第五节介绍了相关文献。第六节描述了经验评估设置，并提供了一个实验比较最流行的动态图的DGN。第七节结束了本文。

Static Graph

Spectral Convolution

主要包含了GCN，表现形式为
$$
\mathbf{x}_u^{\ell+1}=\sigma\left(\Theta_0\mathbf{x}u^\ell+\Theta_1\sum{v\in\mathcal{N}_u}\frac{\mathbf{x}_v^\ell}{\sqrt{\mathbf{deg}(v) \mathbf{deg}(u)}}\right)
$$
主要参考光谱卷积的方式

Spatial Convolution

代表为MPNN，其包含聚合和更新两部分
$$
\mathbf{x}_u^{\ell+1}=\phi_U\left(\mathbf{x}u^\ell,\bigoplus{v\in\mathcal{N}_u}\phi_M\left(\mathbf{x}_u^\ell,\mathbf{x}_v^\ell,\mathbf{e}_\mathrm{uv}\right)\right)
$$
代表有GAT图注意力网络，计算邻居间的注意力系数，进而聚合，不同层之间的传递公式为：
$$
\mathbf{x}u^{\ell+1}=\sigma\left(\sum{v\in\mathcal{N}u}\alpha_\mathrm{uv}\Theta\mathbf{x}v^\ell\right)
$$
为减少图邻接矩阵的复杂度，由于大量图邻接矩阵都全为1，提出了GraphSAGE，随机采点获取随机邻居聚合，传递公式为：
$$
\mathbf{x}u^{\ell+1}=\sigma\left(\Theta\cdot\left[\mathbf{x}u^\ell\parallel\bigoplus{v\in\mathcal{N}S(u)}\mathbf{x}v^\ell\right]\right)
$$
模型聚合邻居表示以计算节点嵌入的方式影响DGN的区分能力。Xu等人[23]表明，大多数DGN最多与1-Weisfeiler-Lehman检验[24]一样强大。特别是，图同构网络（GIN）[23]已被证明与1-Weisfeiler-Lehman测试一样强大，通过计算节点表示为
$$
\mathbf{x}{u}^{\ell+1}=\mathrm{MLP}\Bigg((1+\gamma)\mathbf{x}{u}^{\ell}+\sum{v\in\mathcal{N}{u}}\mathbf{x}{v}^{\ell}\Bigg)
$$
最近，图的表示学习领域的进步引入了新的架构，在DGN和常微分方程（ODE）的域之间建立了联系，主要目标是优化消息传递的各个方面。这些新方法利用ODE的内在属性来提高DGN内消息传递的效率和有效性。通过将图中的信息传播公式化为ODE系统，这些架构有效地解决了多个挑战，例如保持长距离依赖性[25]，降低消息传递的计算复杂性[26]，[27]以及减轻过度平滑现象[28]，[29]。

random walks

学习节点嵌入的另一种策略，包括图的局部和全局属性，依赖于随机游走。随机游走是连接节点序列的随机边序列。Perozzi等人。[30]提出了DeepWalk，这是一种通过将随机游走建模为句子的等价物来学习连续节点嵌入的方法。具体来说，该方法为图中的每个节点采样指定长度的多个遍历，然后利用SkipGram模型[31]基于遍历更新节点表示，将遍历视为句子，将节点表示视为其中的单词。

基于此提出Node2Vec策略提取节点特征。

Dynamic Graph

主要分为离散时间动态图（discrete-time dynamic graphs (D-TDGs)）和连续时间动态图（continuous-time dynamic graphs）。他们的区别如下图所示：

D-TDGs

文献中提出的大多数模型都可以概括为静态DGN和RNN的组合。特别是，一些方法采用堆叠架构，其中顺序使用DGN和RNN，从而能够分别对空间和时间动态进行建模。其他方法将DGN集成在RNN中，允许联合捕获图中的时间演化和空间依赖性。

Spatiotemporal Graphs

解决在给定状态历史Xt的情况下预测下一步骤Xt+1处的节点状态的问题。此种类型问题是没有考虑图之间的变化。

stacked architectures

图卷积循环网络（GCRN），使用切比雪夫谱卷积和Peephole-LSTM方式：
$$
\mathbf{X}{t}^{\prime}=\mathrm{Cheb}(\mathbf{X}{t},\mathcal{E},k,\Theta)\\mathbf{H}{t}=\mathrm{peephole-LSTM}(\mathbf{X}{t}^{\prime})
$$
可以理解为：
$$
\mathbf{X}{t}^{\prime}=\mathrm{DGN}(\mathbf{X}{t},\Theta)\\mathbf{H}{t}=\mathrm{RNN}(\mathbf{X}{t}^{\prime}).
$$

Integrated Architectures

Seo等人。[36]提出了第二个版本的GCRN，它利用这种类型的架构，通过将Chebyshev谱卷积嵌入Peephole-LSTM中。表示为：
$$
\hat{h}=\sigma\left(\mathrm{Cheb}(\mathbf{X}{t},\Theta{x})+\mathrm{Cheb}(\mathbf{H}{t-1},\Theta{h})+\theta_{c}\odot c_{t-1}\right)
$$

General D-TDGs

与时空图不同，一般D-TDG的拓扑结构可以随时间演化。在这种情况下，仅仅利用唯一的节点状态可能会导致性能低下，因为新的拓扑结构会导致图中的不同动态。

解决方法就是把邻接矩阵A take into consideration，会发现其实节点特征没有考虑

该模型的解码器部分是一个多层感知器（MLP），它利用编码器生成的嵌入来预测未来邻接矩阵At+1中每个边缘的概率。

同时存在LRGCN这种聚合边权知识后引入LSTM的模型。

为加速进程，也提出了DynGESN（不知道这个好处在哪。。文章没细说）：

\mathbf{h}{t}^{u}=(1-\gamma)\mathbf{h}{t-1}^{u}+\gamma\mathrm{tanh}\left(\Theta_{i}\mathbf{h}{t}^{u}+\sum{v\in\mathcal{N}{u}^{\prime}}\Theta{r}\mathbf{x}_{t-1}^{v}\right)
$$

Stacked Architectures

和DGN-LSTM类似，其多考虑了上一层对下一层的影响，其MPNN-LSTM表达为：
$$
\begin{aligned}&\tilde{\mathbf{H}}{t}^{\ell}=\mathrm{DGN}^{\ell}\big(\tilde{\mathbf{H}}{t}^{\ell-1}\big)\&\mathbf{H}{t}^{\ell}=\mathrm{UPDATE}\big(\tilde{\mathbf{H}}{t}^{\ell},\mathbf{H}_{t}^{\ell-1}\big)\end{aligned}
$$

Meta Architectures

我们将Meta架构称为学习将图的演变映射到所采用的DGN的参数的演变的函数的方法。解决时不时节点出现消失的特定情况，使用RNN更新GCN的参数。提出了E-GCN，RNN只根据t时间的特征训练t-1时间的GCN参数，GCN用以训练得到的参数训练GCN？（这个不是很明白，要问）
$$
\begin{matrix} \Theta _ { t } = \mathrm{G R U }( X _ { t } , \Theta_ { t - 1 } ) \ H _ { t } = \mathrm{G C N }( X _ { t } , \varepsilon _ { t } , \Theta_ { t } ) \end{matrix}
$$

C-TDGS

由于时间变换，图embedding可能会改变，所以使用事件发生时更新图结构的方式更新。（解决Staleness问题）

Integrated Architectures

Kumar等人。提出了JODIE，一种学习嵌入轨迹以克服陈旧问题的方法。更新节点间的联系时，使用类似RNN的方法。
$$
\widehat { x } _ { u } ( t ) = ( 1 + w ) \odot x _ { u } ( t _ { u } ^ { - } )
$$
其中$w = \Theta _ { p } \Delta t$，$$ \Delta t = t - t _ { u } ^ { - }$$。

后面有一个PP的建模，对于每一个节点，其embedding计算为：
$$
h _ { u } ( t ) = \sigma ( \Theta _ { i } h _ { u } ^ { l o c } ( t ^ { - } ) + \Theta _ { r } h _ { u } ( t _ { u } ^ { - } ) + \Theta _ { e } ( t - t _ { u } ^ { - } ) )
$$
其中第一项是邻居项更新，其表达为：
$$
h _ { u } ^ { l o c } ( t ^ { - } ) = \max ( \left{ \sigma ( \alpha _ { u v } ( t ) \cdot h _ { v } ( t ^ { - } ) ) | v \in N _ { u } ^ { t } ) \right} )
$$
使用注意力机制聚合，聚合后，使用PP建模，结果为：
$$
\lambda _ { u v } ^ { k } ( t ) = f _ { k } ( g _ { u v } ^ { k } ( t ^ { - } ) )
$$

$$
f _ { k } ( z ) = \psi _ { k } \log ( 1 + e x p ( \frac { z } { \psi _ { k } } ) )
$$

$$
g _ { u v } ^ { k } ( t ^ { - } ) = \omega _ { k } ^ { T } \cdot \left[ x _ { u } ( t ^ { - } ) | | x _ { v } ( t ^ { - } ) \right]
$$
